Machine learning supersymmetry¶

Objective: Build an ML model to predict supersymmetric signals.

We will use the SUSY dataset available from the UCI Machine Learning Repository.

From the dataset description:

The data has been produced using Monte Carlo simulations. The first 8 features are kinematic properties measured by the particle detectors in the accelerator. The last ten features are functions of the first 8 features; these are high-level features derived by physicists to help discriminate between the two classes. There is an interest in using deep learning methods to obviate the need for physicists to manually develop such features. Benchmark results using Bayesian Decision Trees from a standard physics package and 5-layer neural networks and the dropout algorithm are presented in the original paper. The last 500,000 examples are used as a test set.n about your data set.

The first column is the class label (1 for signal, 0 for background), followed by the 18 features (8 low-level features then 10 high-level features): lepton 1 pT, lepton 1 eta, lepton 1 phi, lepton 2 pT, lepton 2 eta, lepton 2 phi, missing energy magnitude, missing energy phi, MET_rel, axial MET, M_R, M_TR_2, R, MT2, S_R, M_Delta_R, dPhi_r_b, cos(theta_r1). For detailed information about each feature see the original paper.

References:

  • Baldi, P., P. Sadowski, and D. Whiteson “Searching for Exotic Particles in High-energy Physics with Deep Learning.” Nature Communications 5 (July 2, 2014)

Get the data¶

We import the data. However, we'll import only 10k samples. My tests reveal that this is enough to get good performance and results. Using more than this is computationally quite expensive for what I want to do here.

Since we are using only a small subset of all the data (out of a total of 5e6 datapoints) we expect our performance to be a conservative bound on what can be achieved with the full dataset.

Data set is quite clean and balanced, so we focus on the model building.

def get_probab_and_score(model,X=X_train,y=y_train,T=X_test):
    """Calculates and returns probablities and scores of a given classifier. """
    model.fit(X,y)
    score=model.score(X,y)
    probabilities = model.predict_proba(T)
    return (probabilities,score)

        
def plot_PrecisionRecall_vs_threshold(model,X=X_train,y=y_train, yT=y_test):
    fig, ax = plt.subplots()    
    probabilities,_=get_probab_and_score(model,X,y)
    precision, recall, thresholds = precision_recall_curve(yT, probabilities[:, 1]) 
    pr_auc = metrics.auc(recall, precision)
    plt.title("Precision and Recall vs Threshold")
    plt.plot(thresholds, precision[: -1], "b--", label="Precision")
    plt.plot(thresholds, recall[: -1], "r--", label="Recall")
    plt.ylabel("Precision, Recall")
    plt.xlabel("Threshold of identification")
    plt.legend(loc="lower left")
    plt.ylim([0,1])
    plt.show()    
    return 

def plot_ROC(model,X=X_train,y=y_train, yT=y_test):
    probabilities,_=get_probab_and_score(model,X,y)
    fpr, tpr, thresholds = roc_curve(yT,probabilities[:, 1]) 
    auc=roc_auc_score(yT, probabilities[:, 1])
    fig, ax = plt.subplots()
    plt.title(f'ROC curve (area={auc:.2f})')
    plt.xlabel('False Positive Rate')
    plt.ylabel('True Positive Rate')
    plt.plot([0, 1], [0, 1], linestyle='--')
    plt.plot(fpr, tpr, marker='.')
    plt.show()
    return 


def plot_Learningcurve(model,X=X_train,y=y_train,scoring='roc_auc'):
    train_sizes, train_scores, test_scores = learning_curve(model, X, y, cv=5, scoring=scoring,train_sizes=np.linspace(.05, 1.0, 10))
    train_scores_mean = np.mean(train_scores, axis=1)
    train_scores_std = np.std(train_scores, axis=1)
    test_scores_mean = np.mean(test_scores, axis=1)
    test_scores_std = np.std(test_scores, axis=1)

    fig, ax = plt.subplots()
    plt.fill_between(train_sizes, train_scores_mean - train_scores_std,
                         train_scores_mean + train_scores_std, alpha=0.1,
                         color="r")
    plt.fill_between(train_sizes, test_scores_mean - test_scores_std,
                         test_scores_mean + test_scores_std, alpha=0.1, color="g")
    plt.plot(train_sizes, train_scores_mean, 'o-', color="r",
                 label="Training score")
    plt.plot(train_sizes, test_scores_mean, 'o-', color="g",
                 label="Cross-validation score")

    plt.xlabel("Training examples")
    plt.ylabel("Score")
    plt.legend(loc="best")
    plt.legend(loc="lower right")
    plt.grid("on") 
    plt.show()
    return



def NKfolding_finetuning(model,X=X_train,y=y_train,scoring='roc_auc',N=10,k=5,**ParamsToTune):
    """Makes N times K-fold cross validation on the training set and returns the mean
    of the tuned parameters, and the mean of the score and its std deviation."""
    params=[]
    scores=[]
    for i in range(N):
        clf = GridSearchCV(model,ParamsToTune,cv=k,scoring=scoring)
        clf.fit(X,y)
        params.append(clf.best_params_)
        scores.append([clf.best_score_,clf.cv_results_["std_test_score"][clf.best_index_]])
    pdparams=pd.DataFrame(params)
    best_score=np.mean(scores,axis=0)
    return (dict(pdparams.mean()),best_score[0],best_score[1])

def simple_finetuning(model,X=X_train,y=y_train,scoring='roc_auc',**ParamsToTune):
    """Implements a 5-folded gridSearch on a given classifier. 
    Returns (BestFitParams, BestScore, STD deviation from BestScore)"""
    clf = GridSearchCV(model,ParamsToTune,cv=5,scoring=scoring)
    clf.fit(X,y)
    return  (clf.best_params_,clf.best_score_,clf.cv_results_["std_test_score"][clf.best_index_]) 
  
def NKfolding(model,X=X_train,y=y_train,N=10,k=5,scoring='roc_auc'):
    """Makes N times K-fold cross validation on the data and returns the mean
    of the score and its std deviation."""
    scores=[]
    for i in range(N):
        scores.append(np.mean(cross_val_score(model, X, y, cv=k,scoring=scoring)))       
    return (np.mean(scores))

Find the best classifier¶

After an initial analysis with logistic regression to get a feeling of the problem and set our base-line, we notice that the accuracy expectation is about 77% and AUC $\approx$ 0.8. This is remarkable as the best result obtained wit the deep-learning (DL) model of the paper cited in the introduction above is AUC=0.87. So a simple logistic regression algorithm fairs pretty nicely. So we'll push this even further and see what we can achieve without DL.

The procedure is as follows:

  • Split the data into training+test sets(80:20).
  • Define a list of classifiers and their parameters. We'll use:
    • Logistic regression (SGD-optimized)
    • "Logistic regression" with $L_1$ loss function
    • Linear SVC from the LinearSVC module
    • SVM with RBF kernel
    • XGboost
  • For each estimator, do 10 x 5-fold cross-validations on the training set to tune their hyper-parameters.
  • Compare the tuned models using the AUC score (most relevant for our case of signal discrimination).
  • Finally, calculate the predictions of the 'winning' tuned model and calculate its score using the full data set following the same procedure as above for consistency. (It is OK to use the full data in this way since we are not using it tune the model.)

Final assessment¶

We have quantified the performance of our models, tuned their parameters, and estimated their uncertainties. It appears that the procedure of doing 10 times 5-folds cross-validation gives good results. This is the procedure we will adapt for finalizing the model. We can now use the full dataset with one of the models. However, it apperas that Non-linear SVM classification with RBF kernel works best for our task together with the XGboost algorithm although the latter is also much slower than the former.

We will use XGboost to derive some explainable results, as the plot of the feature importance. After which we plot the final learning curve, and ROC ('Signal efficiency' vs. 'Background Rejection in our case).
